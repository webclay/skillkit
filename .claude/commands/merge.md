Follow the merge-branch workflow to complete a pull request.

## Instructions

1. Read `.claude/skills/workflow/merge-branch/SKILL.md` for the full workflow

2. Steps performed:
   - Identify source and target branches
   - Fetch and pull latest from target
   - Merge feature branch with --no-ff
   - Push the merge
   - Optionally delete the feature branch

3. Handle any merge conflicts if they occur
