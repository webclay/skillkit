# High-End Visual Design Agent Skill: Soft Skill

## Core Philosophy

Positions designers as "Principal UI/UX Architects" tasked with generating "$150k+ agency-level digital experiences" rather than generic websites. Output must "exude haptic depth, cinematic spatial rhythm, obsessive micro-interactions, and flawless fluid motion."

## Key Restrictions (The "Absolute Zero" Directive)

### Typography
Inter, Roboto, Arial, Open Sans, and Helvetica are prohibited in favor of premium alternatives like Geist or Clash Display.

### Visual Elements
Standard thick-stroked icon libraries and generic gray borders are forbidden. The system requires "ultra-light, precise lines" and sophisticated shadow treatment.

### Layout Patterns
Edge-to-edge sticky navigation and symmetrical grid structures without "massive whitespace gaps" are explicitly prohibited.

### Motion
Standard linear or ease-in-out transitions are banned; all motion requires custom cubic-bezier curves simulating "real-world mass and spring physics."

## Design Archetypes

The framework offers three vibe categories:
- Ethereal Glass
- Editorial Luxury
- Soft Structuralism

Paired with three layout approaches:
- Asymmetrical Bento
- Z-Axis Cascade
- Editorial Split

This enables unique combinations while maintaining premium standards.

## Technical Requirements

All animations must use transform and opacity exclusively, avoiding layout-triggering properties. Blur effects apply only to fixed elements, and section padding must reach minimum `py-24` for appropriate breathing room.
